## [v0.2.6.1]

### Changed

- 'Gifts for Roll' now shows on multiple Support Car locations on the map.
- Blue Refractor now shows as static instead of togglable.

### Fixed

- Logic for Underground Ruins checks that can be accessed via Drill Arm.

## [v0.2.6.0] - 2026-03-28

### Added

- Initial Release.
- Support for Archipelago location and item auto-tracking.
- Support for auto-tabbing based on the player's map location.
